# 🌿 Mamba Widget

**"A cute, glassmorphic daily focus widget that brings plant vibes and productivity to your desktop."**

---

## ✨ Features

- **Floating Launcher**: A discreet, pulsing circular icon that stays on your screen for quick access.
- **Glassmorphism UI**: High-end aesthetic with 25px background blur and smooth cubic-bezier transitions.
- **Task Management**: Simple, category-coded task list with a rapid `+` add system and progress tracking.
- **Pomodoro Timer**: A bold "Hero" timer with a pulse animation during active sessions and custom sound alerts.
- **Persistent Notes**: A spacious area for your thoughts that automatically saves as you type.
- **Tray & Shortcut Support**: Fully integrated with the system tray and global hotkeys.
- **Portable & Lightweight**: Run the `EXE` directly without installation—your settings stay with the app.

## 🚀 Getting Started

### Installation (Portable)
1. Download `MambaWidget_Portable.zip`.
2. Extract the folder to your preferred location.
3. Run `MambaWidget.exe`.

### Global Shortcuts
- **Toggle Widget**: `Ctrl + Shift + T`

## 🛠️ Built With
- **Electron**: Cross-platform desktop framework.
- **Vanilla JavaScript/CSS**: For high-performance, pixel-perfect UI.
- **Inter Font**: Clean, modern typography.

## 📝 Usage Tips
- **Toggle Mode**: Use the "Preview" chip to hide input fields and focus solely on your tasks.
- **Categories**: Assign "Study", "Work", or "Personal" to tasks for visual organization.
- **Audio Alerts**: Ensure your system volume is up to hear the `alert.mp3` when your timer or task reminder fires.
- **Test Button**: Use the "Test Sound & Notify" button in the Timers tab to verify your system setup.
